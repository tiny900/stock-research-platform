AI-Powered Stock Research Platform

Multi-agent stock analysis with RAG knowledge retrieval and systematic prompt engineering

CrewAI RAG ChromaDB Prompt Engineering Streamlit Python
View on GitHub See Demo Watch Video

Key Features

Multi-Agent Orchestration

4 specialized CrewAI agents (Controller, Data Collector, Analyst, Report Writer) working in a sequential pipeline to produce comprehensive stock research reports.

RAG Knowledge Retrieval

ChromaDB-powered vector store with 100+ document chunks from SEC filings, earnings transcripts, financial glossary, and market analysis reports. Semantic search with ticker-based re-ranking.

Prompt Engineering Framework

3-version prompt system (Basic, Structured, CoT+RAG) with composable strategies: chain-of-thought reasoning, few-shot examples, RAG context injection, and output schema constraints.

5 Technical Indicators

Custom-built technical analysis tool calculating RSI, MACD, SMA, EMA, and Bollinger Bands with signal generation and consensus assessment.

Interactive Web UI

Streamlit dashboard with 5 tabs: Analysis Dashboard (Plotly charts), RAG Context viewer, Report generator, Prompt Engineering Lab, and System Metrics.

Demo Mode

Full functionality without API keys using synthetic data fallback, pre-computed ChromaDB embeddings, and TF-IDF retrieval backup.

Quick Start

# Clone and install
git clone https://github.com/TianyuZhang-NEU/stock-research-platform.git
cd stock-research-platform
pip install -r requirements.txt

# Run demo (no API keys needed)
python main.py --demo

# Launch web UI
streamlit run app.py

With LLM Integration

# Configure API key
cp .env.example .env
# Edit .env with your API key

# Full analysis
python main.py --symbol AAPL

# With specific prompt version
python main.py --symbol TSLA --prompt v2_structured
